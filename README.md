## informational-affairs

A likeness of the website <a href="https://informationalaffairs.com/" target="_blank">Informational Affairs</a> by <a href="https://folder.studio/" target="_blank">Folder Studio</a>. Made with the intent of learning how to create a fixed title bar that can expand or collapse with a click as well as how to render a list of tags and filter the content displayed when a tag is clicked.
